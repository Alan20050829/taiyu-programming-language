## 📜 **使用方式**
1. 在 C / C++ 程式中：
   ```c
   #include "taiyu_c.h"  // C
   #include "taiyu_cpp.hpp"  // C++
   ```

2. 在 Python 程式中：
   ```python
   from taiyu_py import *
   ```
